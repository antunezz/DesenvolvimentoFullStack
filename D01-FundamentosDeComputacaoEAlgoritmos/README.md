## Fundamentos de computação e algoritmos

### Algoritmos de ordenação

* Bubble sort
  ```sh
  node bubblesort <tamanho> <min> <max>
  ```

* Insertion sort
  ```sh
  node insertionsort <tamanho> <min> <max>
  ```

* Merge sort
  ```sh
  node mergesort <tamanho> <min> <max>
  ```

* Quick sort
  ```sh
  node quicksort <tamanho> <min> <max>
  ```
